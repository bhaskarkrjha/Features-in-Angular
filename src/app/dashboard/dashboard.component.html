Welcome to the Angular UI Components

This application demonstrates various UI components and patterns implemented in Angular. Below is an overview of the features you can explore:

Standard Table

This section displays a standard HTML table using <tr><th><td> elements, styled with Angular Material for a modern look and feel.

Ngx-data-table

Explore a powerful data table built with the ngx-data-table library, offering features like sorting, filtering, and pagination.

Table with Search and Actions

This section showcases an ngx-datatable integrated with search functionality and action.

Dropdowns for filter

This section demonstrates how to implement a dropdown menu for filtering data.

Email list

This section demonstrates how to implement a field that store multiple email Ids.

Reactive Forms

Learn about Reactive Forms in Angular, providing a robust way to handle form validation and dynamic form creation.

Cron Job Scheduler

Implement a cron job scheduler UI, where users can configure recurring tasks using an intuitive interface.

Query Builder

Explore a query builder tool that allows users to create complex queries with a visual interface, simplifying data retrieval processes.
